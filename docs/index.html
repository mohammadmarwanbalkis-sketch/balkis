const incomeTax = defineCalculation({
  id: "payroll.income-tax",
  version: "1.0.0",
  summary: "Flat 20% income tax on gross salary.",
  input: z.object({}),
  output: z.object({ tax: z.number() }),
  dependencies: [grossSalary],     // typed, refactor-safe, cycle-free
  calculate: ({ deps }) => ({
    tax: deps["payroll.gross-salary"].gross * 0.2,
  }),
});

🔒 Validated boundaries

Zod schemas guard every input and output. Nothing unvalidated flows anywhere.

🎯 Deterministic

Same inputs ⇒ same outputs and trace. Determinism checks catch Math.random leaks in CI.

🧾 Audit trail built in

Every run records every step: inputs, outputs, fired rules, durations.

⚖️ Rules as JSON

Condition trees with priorities — storable, diffable, generatable by agents.

🔮 Scenarios & Monte Carlo

What-if overlays, sensitivity sweeps, seeded simulations. All deterministic.

💰 Exact decimal math

bigint fixed-point with banker's rounding. 0.1 + 0.2 = 0.3, finally.

⚡ Parallel, measured

Up to 56.8× on async fan-ins — with the benchmark to prove it.

🤖 AI-native metadata

The whole catalog as JSON Schema. Agents modify logic without reading implementations.
